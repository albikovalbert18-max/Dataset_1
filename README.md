# О проекте

В рамках курсового проекта выполнен поиск и первичный анализ наборов данных различных типов.  
Глава 1 посвящена анализу **табличных данных** на примере датасета Banking Dataset for Marketing Targets.

Цель анализа:
- изучить структуру и качество данных;
- выявить пропуски, выбросы, дубликаты;
- провести визуализацию и корреляционный анализ;
- сформулировать выводы о пригодности данных для задач машинного обучения.

# Датасет

**Название:** Banking Dataset for Marketing Targets  
**Источник:** [Kaggle — Banking Dataset](https://www.kaggle.com/datasets/prakharrathi25/banking-dataset-marketing-targets)  
**Лицензия:** CC BY 4.0 (учебное использование разрешено)  
**Объём:** 45 211 записей, 17 признаков  
**Файлы:** `train.csv`, `test.csv` (разделены автором датасета)

Файлы данных не включены в репозиторий из-за размера.  
Для запуска кода скачайте `train.csv` и `test.csv` по ссылке выше и положите их в папку с проектом.
